# WhatsApp Sender

Script em Python que lê até três contatos do Supabase e envia pela Z-API:

```text
Olá, <nome_contato> tudo bem com você?
```

## 1. Tabela no Supabase

Execute no SQL Editor:

```sql
CREATE TABLE public.contatos (
    id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    created_at timestamptz DEFAULT now(),
    nome text NOT NULL,
    telefone text NOT NULL UNIQUE
);

ALTER TABLE public.contatos ENABLE ROW LEVEL SECURITY;

GRANT SELECT ON TABLE public.contatos TO anon;

CREATE POLICY "Permitir leitura de contatos"
ON public.contatos FOR SELECT
TO anon
USING (true);

INSERT INTO public.contatos (nome, telefone) VALUES
    ('João', '5511999999999'),
    ('Maria', '5521999999999'),
    ('Pedro', '5531999999999');
```

O telefone deve conter DDI, DDD e número, somente com dígitos.

## 2. Variáveis de ambiente

```bash
cp .env.exemplo .env
```

Preencha o `.env`:

```env
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_KEY=sua-chave-anon
ZAPI_INSTANCE_ID=sua-instancia
ZAPI_TOKEN=seu-token
ZAPI_CLIENT_TOKEN=seu-client-token
```

## 3. Instalação

```bash
python -m venv .venv
source .venv/bin/activate
pip install .
```

## 4. Execução

```bash
# Simula sem enviar
python principal.py --teste

# Envia para até três contatos
python principal.py
```

Com Docker:

```bash
docker compose run --rm whatsapp-sender --limite 3
```
